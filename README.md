LotusAdmin2.0其主要特性包括：

 + 基于ThinkPHP5.0.12+Layui2.0+adminLTE+BootSratp3.0
 + 高颜值，不将就
 + 完整的权限系统
 + 极简不冗杂，无需删除代码即可立刻开始定置自己的后台系统.



LotusAdmin 2.0的运行环境要求PHP5.4以上。安装方法如下:

 + 1.设置虚拟域名到public
 + 2.导入数据库lotus.sql
 + 3.去掉数据库配置文件./application  的database.php.bak的.bak后缀，并修改自己对应的数据库配置（群下载忽略）
